## 租貸超跑 形象網站

使用 Laravel 8 開發，以Laravel-admin 作為後台管理，功能如下:

- FullPage 響應式選單
- SEO 重要資訊 ｜ 後台操作控制
- 聯絡我們 （驗證碼防止機器人惡意操作）
- 廣告橫幅輪播 ｜ 後台操作控制
- 超跑＋性能車款 ｜ 後台操作控制
- 超跑展示輪播 ｜ 後台操作控制
- 租車流程＋須知 ｜ 後台操作控制
- 最新優惠 ｜ 後台操作控制
- 右下角小圖示 ｜ 社群聯絡以及電話、回到頂端
- Page 加載動畫
- Sweetalert 消息提醒
